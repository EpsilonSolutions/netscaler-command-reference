#configstatus

The following operations can be performed on "configstatus":


##show configstatus

Display status of packet engines.


##Synopsys

show configstatus


##Outputs

<b>consistent</b>
State of packet engines.

<b>culpritCore</b>
Culprit core id.

<b>core</b>
Core id.

<b>culpritCoreConfString</b>

<b>coreConfString</b>

<b>devno</b>

<b>count</b>

<b>stateflag</b>



##Example

show configstatus
