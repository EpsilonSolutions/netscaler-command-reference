#rise rhi

The following operations can be performed on "rise rhi":


##show rise rhi

Retrieves RISE RHI rules programmed


##Synopsys

show rise rhi


##Arguments

<b>summary</b>

<b>fullValues</b>



##Outputs

<b>IPAddress</b>
(V)IP advertised

<b>prefixLen</b>
Prefix Length

<b>hostRtGw</b>
Gateway for the advertised IP

<b>nextHopVlan</b>
Vlan id on which the gateway is present

<b>weight</b>
Cost of the route

<b>vserverRHILevel</b>
Advertisement level

<b>devno</b>

<b>count</b>

<b>stateflag</b>
